# Angular: Reusable orderBy component using Observables #

## How-to use ##
### 1. Installation ###
```
npm install angular-sorting-component
```

Import the module where needed.

```typescript
import { Ng2SortModule } from 'angular-sorting-component';

@NgModule({
  imports: [

    Ng2SortModule
  ]
})
export class AppModule { }
```
